# Repo Habitat Plan (Phase 2)

Repo Habitat is the product pivot for GH Tracker.

It combines three layers:
- Local Git machine stats from the collector snapshots.
- ReleaseBar-inspired repo health placeholders (release/CI/PR/issues sync states).
- Original Slimy.ai retro virtual-pet inspired repo companions.

Phase 2 scope:
- Add health and pet contracts.
- Add deterministic pet generation from stable repo identity.
- Add dashboard habitat section and placeholder pixel pet rendering.
- Keep GitHub API sync out of scope.

Out of scope in Phase 2:
- GitHub API token sync.
- Cron/systemd timers.
- Final production sprite sheets.

Principles:
- Pets reward healthy maintenance and consistency, not commit spam.
- Pets are original assets; no official Tamagotchi characters, sprites, or naming.
- Health can use local signals now and remote sync signals later.
