# Visual Style

## Direction
Cyber-retro Slimy.ai dashboard:
- deep black and violet base gradients
- neon magenta borders and glow accents
- lime green highlights for positive activity
- monospace labels + futuristic heading font
- subtle scanline and grid overlays

## UX Principles
- machine-first visibility, then combined rollup
- high-density telemetry cards without clutter
- charts and timeline in equal prominence
- always show system/debug state for operator confidence
